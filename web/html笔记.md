### tag 标签
由尖括号包围的关键词，eg：`<html>`

### 中文编码
直接输出中文可能会出现乱码，故需在头部声明为GBK/utf-8
`<meta charset="UTF-8">`

**保存 html 文件时，文档编码和 meta 设置的编码，一定要相同，只要不相同，就一定会出现乱码！**

### 基础
1. 标题-通过`<h1> - <h6>` 标签来定义的
2. 段落-通过标签 `<p>` 来定义的
3. 链接是通过标签 `<a>` 来定义的
4. 图像是通过标签 `<img>` 来定义的.

### 属性
通常出现在 HTML 标签的开始标签中

### 转义
特殊字符（如&）在html中有特殊意义，所以需要对这些字符进行转意，否则html会将其处理为特殊意义。
`& -> &amp;`
